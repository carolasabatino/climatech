# ⛅ CLIMATECH

Proyecto desarrollado junto a un equipo de 10 estudiantes de Argentina, Paraguay, Brasil y Uruguay durante una Hackathon internacional del Mercosur. El objetivo fue desarrollar un prototipo de estación meteorológica capaz de medir distintas variables climáticas. Eso lo hacía mediante sensores de lluvia, sensación térmica, viento, temperatura y humedad conectados a Arduino.
Los datos recolectados se almacenaban en una base de datos y luego se mostraban en un sitio web. Dentro del equipo de informática, me encargué junto a otro compañero del desarrollo del sitio web utilizando React, JavaScript, HTML y CSS, creando esta interfaz para visualizar los datos obtenidos por el sistema.
